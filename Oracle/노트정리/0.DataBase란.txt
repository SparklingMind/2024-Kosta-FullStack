
DataBase란?
  : 데이터를 저장하는 공간


RDBMS : Relation DataBase Management System 
  => 관계형(PK + FK ) 데이터 베이스 

 : 데이터를 저장할 때 관계형데이터베이스 이론(개념적설계 -> 논리적설계 -> 물리적설계 )을 적용하여 체계적으로 데이터를 저장함으로써
    1) 데이터중복의 최소화
    2) 정합성의 문제를 해결(삽입, 삭제, 수정 이상현상을 제거 ) 
    3) 데이터무결성원칙에 위배 되지 않는다.
    4) 여러 유저들간의 공유 
    5) 데이터 조작을 손쉽게 할수 있다.(CRUD 작업)





 DBMS의 종류
  1) Oracle
  2) Ms_SQL
  3) My-SQL
  4) H2
  5) Maria DB
